# ozip2img
Convert oppo ozip (.ozip) into filesystem ext4 image

## Requirements
```
Brain
Brotli
Linux
Python
Ozip file to convert
```

## How to use

### Download tools
```
git clone --recurse-submodules https://github.com/SiddharthBharadwaj/ozip2img.git
cd ozip2img
```
### Using
```
./ozip2img.sh or bash ozip2img.sh
```
